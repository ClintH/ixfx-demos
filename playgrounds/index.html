../ playgrounds

Playgrounds are for trying things out. The code is not meant to be built upon.

  • keys: Key events
  • pointer: Pointer events
  • pointer-pen: Pen pointer events (stylus input required)
  • pointer-gesture: Touch gestures (touch input required)
  • force: Force/pressure data from Apple trackpads (when using Safari) or stylus
  • espruino-repl: Run lines of code on an Espruino
  • espruino-plot: Plot data from Espruino
  • espruino-drv2605: Haptic motor
